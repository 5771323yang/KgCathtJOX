# 动漫商城管理系统 java673

## 项目概述

动漫商城管理系统是一个基于Spring Boot和Vue的前后端分离的Java项目，适用于动漫商品的销售与管理。

## 技术栈

- 后端：Spring Boot
- 前端：Vue

## 功能模块

### 系统角色

- 管理员
- 用户

### 管理员功能

- 个人中心
- 用户管理
- 商品类型管理
- 动漫商品管理
- 系统管理
- 订单管理

### 用户功能

- 动漫商品浏览与购买
- 动漫资讯查看
- 个人中心管理
- 购物车管理
- 在线咨询

## 运行环境

- Java运行环境
- Node.js环境（前端项目）

## 部署步骤

（以下步骤为常规部署流程，具体步骤可能根据实际项目配置有所差异）

1. 准备后端运行环境，包括Java环境及数据库等。
2. 部署后端Spring Boot服务。
3. 准备前端运行环境，包括Node.js等。
4. 构建前端Vue项目，并部署到Web服务器。
